# Тестовое задание

## Условия задачи

1. **Закрыть уязвимость GO-2024-2554:**
    - Используйте инструмент `govulncheck` для сканирования уязвимостей в вашем проекте.
    - Найдите и устраните уязвимость `GO-2024-2554` путем написания патча.
    - Дополнительный плюс - при устранении всех других найденных уязвимостей.
    - После исправления убедитесь, что уязвимость устранена, и приложите отчет из `govulncheck`.

2. **Создать мультистейдж сборку Docker:**
    - Напишите `Dockerfile` с использованием мультистейдж сборки.
    - Сборка должна включать:
        - Этап для сборки бинарного файла Go.
        - Этап для минимального образа с готовым бинарным файлом.
    - Размер финального Docker-образа должен быть минимальным (рекомендуется использовать `scratch` или `alpine`).

## Ожидаемый результат

1. Отчет `govulncheck`, подтверждающий устранение уязвимости.
2. Рабочий `Dockerfile`, позволяющий собрать минимальный образ.
3. Документация и инструкции в `README.md` в репозитории.
4. Корректный вывод приложения при его запуске, подтверждающий, что приложение работает и ничего не сломалось.

## Критерии оценки

- Устранение уязвимости.
- Качество и структура Docker-образа.
- Понятность инструкций в `README.md`.
